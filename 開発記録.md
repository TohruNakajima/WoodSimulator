## TozawaMCPサブモジュール運用ルール

### MCPツール更新時の必須手順
1. Assets/TozawaMCPディレクトリでツール修正
2. サブモジュール内でcommit: `cd Assets/TozawaMCP && git add . && git commit -m "説明"`
3. サブモジュールをpush: `git push origin main`
4. プロジェクトルートに戻り、サブモジュール参照を更新してcommit

### 禁止事項
- サブモジュール外でのMCPツール直接編集
- サブモジュール更新のコミット忘れ
