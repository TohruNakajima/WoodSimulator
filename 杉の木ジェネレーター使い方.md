# 杉の木ジェネレーター 使い方

## 概要

`PineTreeGenerator` コンポーネントをGameObjectにアタッチすることで、プロシージャルに杉の木を生成できます。  
各パラメータはInspector上でスライダーで調整でき、`autoRegenerate` が有効な場合は値を変更するたびに自動で再生成されます。

---

## パラメータ解説

### WhorlCount（枝の段数）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `int` / 8〜32 |
| デフォルト値 | 16 |
| Inspector表示 | Pine Shape > Whorl Count |

**説明**  
幹に生成される枝の「段数」を決定します。`BranchesPerWhorl` と掛け合わされ、木全体の枝の総本数（`WhorlCount × BranchesPerWhorl`）が決まります。

**調整のコツ**
- 値を大きくすると枝が密になり、葉も増えるためボリューム感が出ます
- 値を小さくするとスカスカで枯れた印象の木になります
- 枝の総本数が増えるとメッシュ頂点数も増加するため、パフォーマンスに注意してください

---

### BranchesPerWhorl（1段あたりの枝の本数）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `int` / 1〜16 |
| デフォルト値 | 8 |
| Inspector表示 | Pine Shape > Branches Per Whorl |

**説明**  
1段あたりに生成される枝の本数です。各枝は幹の周囲360°からランダムな方位角で生えます。

**調整のコツ**
- 値を増やすと1段あたりの枝が増え、より密な樹冠になります
- 頂部の枝の本数もこの値に連動します（最低3本）
- `WhorlCount` と合わせて総本数を意識してください（例：16段 × 8本 = 128本）

---

### TrunkHeight（幹の高さ）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `float` / 5〜25（メートル） |
| デフォルト値 | 13 |
| Inspector表示 | Pine Shape > Trunk Height |

**説明**  
木の幹の高さです。枝の配置範囲や頂部の枝の位置もこの値を基準に決まります。

**調整のコツ**
- 若木（樹齢10年程度）: 5〜8
- 成木（樹齢25〜55年）: 10〜18
- 大木（樹齢75年以上）: 18〜25
- 高さを変えた場合は `BaseBranchLength` や `TrunkRadius` も合わせて調整すると自然な比率になります

---

### TrunkRadius（幹の太さ）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `float` / 0.05〜0.5（メートル） |
| デフォルト値 | 0.22 |
| Inspector表示 | Pine Shape > Trunk Radius |

**説明**  
幹の根元の半径です。幹は根元から先端に向かって細くなります（テーパー形状）。枝の生え際の位置にも影響し、枝は幹の表面付近から生成されます。

**調整のコツ**
- 若木: 0.05〜0.12
- 成木: 0.15〜0.30
- 大木: 0.30〜0.50
- `TrunkTipRadius`（先端の太さ）とのバランスも重要です

---

### BaseBranchLength（根元付近の枝の長さ）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `float` / 1〜7（メートル） |
| デフォルト値 | 3.8 |
| Inspector表示 | Branch Settings > Base Branch Length |

**説明**  
幹の下部（根元側）に生える枝の基本の長さです。枝の長さは幹の下部から上部にかけて `BaseBranchLength` → `TipBranchLength` へ線形に補間されます。さらに各枝ごとに0.7〜1.3倍のランダムなスケールが適用されます。

**調整のコツ**
- 大きくすると下部の枝が長くなり、樹冠が広がります
- `TipBranchLength` との差が大きいほど、円錐形のシルエットになります
- `TrunkHeight` の1/3〜1/2程度が自然な見た目の目安です

---

### TipBranchLength（先端付近の枝の長さ）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `float` / 0.1〜4（メートル） |
| デフォルト値 | 1.1 |
| Inspector表示 | Branch Settings > Tip Branch Length |

**説明**  
幹の上部（先端側）に生える枝の基本の長さです。頂部の枝（crown部分）はこの値の70%がベースになります。

**調整のコツ**
- `BaseBranchLength` より小さい値にすると自然な円錐形になります
- 両者を近い値にするとずんぐりした樹形になります
- 0.5以下にすると先端が非常に細くなり、尖った印象になります

---

### BranchStartHeight（枝の生え始め位置）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `float` / 0〜0.5（0=根元、0.5=幹の半分の高さ） |
| デフォルト値 | 0.09 |
| Inspector表示 | Branch Settings > Branch Start Height |

**説明**  
幹のどの高さから枝が生え始めるかを、0〜1の正規化値で指定します。0が根元、1が頂点です。枝はこの値と `BranchEndHeight`（デフォルト: 0.95）の範囲内にランダムに配置されます。

**調整のコツ**
- 0に近い値: 根元付近から枝が生え、森の中の若木のような見た目
- 0.3〜0.5: 下部が枝なしの幹だけになり、成熟した杉や街路樹のような見た目
- 下枝が落ちた老木を表現したい場合は値を大きくしてください

---

### BaseLeavesPerBranch（1枝あたりの葉の枚数）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `int` / 16〜64 |
| デフォルト値 | 36 |
| Inspector表示 | Leaves (Card) > Base Leaves Per Branch |

**説明**  
幹の下部の枝1本あたりに生成される葉カードの基本枚数です。上部に行くほど枚数が減少し、最上部では約45%まで減ります。頂部（crown）の枝には約50%の枚数が適用されます。

**調整のコツ**
- 値を大きくすると葉が密になり、鬱蒼とした印象になります
- 値を小さくすると枝が見え、冬枯れや若木のような印象になります
- 枝の総本数 × この値 が総葉枚数に直結するため、パフォーマンスへの影響が大きいパラメータです

---

### LeafCardLength（葉カードの長さ）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `float` / 0.2〜2（メートル） |
| デフォルト値 | 0.72 |
| Inspector表示 | Leaves (Card) > Leaf Card Length |

**説明**  
個々の葉を表現する矩形カード（ポリゴン板）の長さです。生成時に±3%のランダムな揺らぎが加わります。

**調整のコツ**
- 大きくすると1枚の葉が広い面積をカバーし、少ない枚数でもボリュームが出ます
- 小さくすると繊細な葉の表現になりますが、密度を出すには `BaseLeavesPerBranch` を増やす必要があります
- リーフマテリアルのテクスチャサイズに合わせて調整してください

---

### LeafCardWidth（葉カードの幅）

| 項目 | 内容 |
|------|------|
| 型・範囲 | `float` / 0.08〜1（メートル） |
| デフォルト値 | 0.23 |
| Inspector表示 | Leaves (Card) > Leaf Card Width |

**説明**  
個々の葉カードの幅（短辺）です。`LeafCardLength` と同様に±3%のランダムな揺らぎが加わります。

**調整のコツ**
- 杉の葉は細長いため、`LeafCardLength` に対して1/3〜1/4程度の幅が自然です
- 幅を広くすると広葉樹のような印象になります
- 極端に小さくすると針のような見た目になります

## パラメータ早見表

| パラメータ | 範囲 | デフォルト | 影響 |
|-----------|------|-----------|------|
| WhorlCount | 8〜32 | 16 | 枝の段数（密度） |
| BranchesPerWhorl | 1〜16 | 8 | 1段の枝本数 |
| TrunkHeight | 5〜25 | 13 | 木の高さ |
| TrunkRadius | 0.05〜0.5 | 0.22 | 幹の太さ |
| BaseBranchLength | 1〜7 | 3.8 | 下部の枝の長さ |
| TipBranchLength | 0.1〜4 | 1.1 | 上部の枝の長さ |
| BranchStartHeight | 0〜0.5 | 0.09 | 枝の生え始め位置 |
| BaseLeavesPerBranch | 16〜64 | 36 | 葉の枚数/枝 |
| LeafCardLength | 0.2〜2 | 0.72 | 葉の長さ |
| LeafCardWidth | 0.08〜1 | 0.23 | 葉の幅 |
